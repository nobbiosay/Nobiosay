KWS — Versi Minimal untuk Vercel

Vercel tidak mendukung PHP secara native. Paket ini mengubah proyek menjadi static site yang siap dideploy.

Cara Deploy

  1. Ekstrak zip ini, lalu push ke repository Git (GitHub/GitLab/Bitbucket).
  2. Buat proyek di Vercel > Import Project > pilih repo.
  3. Framework preset: Other. Output directory: /.
  4. Deploy. Selesai ✅

Menggunakan Data Nyata

Ubah data.json agar mengarah ke API Anda (misal dari layanan lain), atau ganti fetch pada index.html untuk memanggil endpoint.

Opsi Backend

  • Porting PHP ke Node/Edge Functions di Vercel (disarankan).
  • Menjalankan backend PHP di layanan lain (Railway/Render/Hostinger) dan mengarahkan front-end ini ke API tersebut.

Struktur

/
├─ index.html
├─ data.json
└─ assets/
   ├─ custom.css
   └─ logo.png (opsional)
